<実装する機能>
・チームによる開発ができるタスク管理アプリケーション

<画面遷移>
*エラーメッセージはflashスコープに保存する。
*ログイン状態
ユーザー単位であるsessionに Consts.LOGIN= uid を保存する。
データベースアクセスはこのuidを取得して行う。

[ログインページ]
・新規ユーザー登録の場合 => ユーザー登録ページへ
・uid、pwによりユーザー認証。ログイン状態にする。 => トップページへ

[ユーザー登録ページ︎]
・uid、pw、nameを登録。ログイン状態にする。 => ユーザー登録結果ページへ
・戻るも選択可。 => ログインページへ

[ユーザー登録結果ページ] -> 削除
・uid、nameを表示。 => トップページへ
・ログアウトも選択可。ログイン状態解除 => ログインページへ

[トップページ]
・タスクの一覧表示。(publicとprivateで分ける)
・タスクの編集を行える。 =>  タスク編集ページへ
・マイページへ => マイページへ
・ログアウト => ログインページへ

[タスク編集ページ]
・一覧表示から直接編集できるのがベスト
	*編集の権限をチェック

・質問すること
urlから直入力でのリクエストを禁止する方法。
->E13を参照
