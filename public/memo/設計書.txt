<実装する機能>
・チームによる開発ができるタスク管理アプリケーション

<画面遷移>
[ログインページ]
・新規ユーザー登録の場合 => ユーザー登録ページへ
・uid、pwによりユーザー認証。ログイン状態にする。 => トップページへ

[ユーザー登録ページ︎]
・uid、pw、nameを登録。ログイン状態にする。 => ユーザー登録結果ページへ
・戻るも選択可。 => ログインページへ

[ユーザー登録結果ページ]
・uid、nameを表示。 => トップページへ
・ログアウトも選択可。ログイン状態解除 => ログインページへ

[トップページ]
・タスクの一覧表示。(publicとprivateで分ける)
・タスクの編集を行える。 =>  タスク編集ページへ
・マイページへ => マイページへ
・ログアウト => ログインページへ

[タスク編集ページ]
・一覧表示から直接編集できるのがベスト
	*編集の権限をチェック
