# exp4todo
task management tool (Java+Play)

## Requirements
- Java 6<=12
- Python 2

## Setup
```
git clone https://github.com/You-Your-Yu/exp4todo.git
cd exp4todo
source setup.sh
```

## Run
```
play run
```
